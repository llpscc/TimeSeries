# Прогнозирование временных рядов — Favorita Grocery Sales

## 1. Описание проекта

Цель проекта — прогнозирование ежедневных продаж (`unit_sales`) для пар  `(store_nbr, item_nbr)` на основе исторических данных и внешних факторов. 

Задача - прогнозирование будущих значений продаж на горизонте 16 дней. 
Для прогнозирования используются исторические признаки временных рядов, календартные признаки, а так же экзогенные.
Были рассмотрены простые baseline модели, модель машинного обучения и модель глубинного обучения.

---

## 2. Описание данных

Основной датасет содержит:

- `date` — дата
- `store_nbr` — идентификатор магазина
- `item_nbr` — идентификатор товара
- `unit_sales` — целевая переменная
- `onpromotion` — наличие акции

Дополнительные данные:

- `items` — характеристики товаров (категория, класс, перишабельность)
- `stores` — информация о магазинах (город, тип, кластер)
- `transactions` — количество транзакций
- `oil` — цена нефти (внешний фактор)

## 4. Валидация

Так как в задачах прогнозирования временных рядов нельзя использовать случайное разбиение данных, для прогнозирования использовался time-based валидация. Задача требует прогнозирование на горизонте 16 дней, поэтому в качестве валидационной выборки были использованы последние 16 дней тренировочной.

Особенности:
- без перемешивания
- без утечки данных
- имитация реального прогноза

Для ML моделей используется **рекурсивный (autoregressive) прогноз**.

---

## 6. Инференс (ML)

Прогноз строится рекурсивно:

1. История инициализируется из train
2. Для каждого следующего дня:
   - считаются лаги
   - делается предсказание
   - результат добавляется в историю

## 7. Метрика

Качество модели оценивалось с использованием метрики NWRMSLE (Normalized Weighted Root Mean Squared Logarithmic Error), которая используется в соревновании Kaggle. 

Она учитывает логарифм продаж и использует веса, заданные в таблице items под признаком perishable. Он показывает является ли товар скоропортящимся. Для скоропортящихся товаров определен вес 1.25, а для не скоропортящихся 1.0
